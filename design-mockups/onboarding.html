Pi Browser Agent

Control your browser with natural language through Pi.

✓
Install Pi CLI
ready
Pi Browser Agent needs the Pi coding agent installed locally.
npm install -g @earendil-works/pi-coding-agent
2
Install Native Host
required
The native host bridges the browser extension and Pi.
curl -fsSL https://pi-browser-agent.dev/install.sh | bash
3
Configure your LLM
required
Either log in through Pi or enter your API key in the extension settings.
pi /login
4
Pin the extension
required
Click the extensions icon in Chrome and pin Pi Browser Agent to your toolbar. Then open any page and click the icon to start.
SECURITY: This extension can read page content and execute browser actions. Enable "Confirm sensitive actions" in settings to review every tool call.
Need help? Read the docs or open an issue on GitHub.
